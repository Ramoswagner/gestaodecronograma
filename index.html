Project Wagner BI Dashboard para Gestores

Configure seu painel em segundos

Conecte uma planilha Google Sheets e monitore projetos com atualização automática.

Use o link de publicação CSV do Google Sheets. Veja a aba "Como configurar" para instruções detalhadas.

1
Abra sua planilha no Google Sheets

Certifique-se de que há duas abas: Informações do Projeto e Atividades.

2
Acesse Arquivo → Compartilhar → Publicar na web

No menu superior da planilha, acesse Arquivo → Compartilhar → Publicar na web.

3
Publique a aba "Atividades" como CSV

Selecione a aba Atividades e o formato Valores separados por vírgula (.csv). Clique em Publicar.

4
Cole o link e conecte

Copie o link gerado, vá na aba Conectar Planilha e cole. O dashboard carrega automaticamente.

A planilha deve ter duas abas: Informações do Projeto (nome, responsável, instituição) e Atividades (dados do projeto).

Aba: Informações do Projeto

Linha 1, Coluna B: Nome do Projeto  →  ex: Sistema de Gestão Municipal
Linha 2, Coluna B: Responsável  →  ex: Ana Lima
Linha 3, Coluna B: Instituição  →  ex: Prefeitura de Belo Horizonte
